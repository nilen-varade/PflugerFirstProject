## Project Context

This repository contains simulations and analysis for a **mass timber high school gymnasium design** by Pfluger Architects. The project focuses on:
- Structural performance analysis of mass timber systems for large-span gym spaces
- Environmental benefits and carbon footprint assessment
- Cost and constructability comparisons with traditional construction
- Feasibility demonstration for institutional athletic buildings

## Project Type

This is an architectural simulation and analysis project that will involve:
- Structural analysis software outputs and data
- Building Information Modeling (BIM) files and exports
- Energy modeling and simulation results
- Cost estimation data and reports
- Technical documentation and analysis reports
